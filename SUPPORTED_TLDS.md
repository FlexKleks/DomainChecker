## Adding Custom TLDs

You can add custom TLDs by editing your `config.json`:

```json
{
  "tlds": [
    {
      "tld": "custom",
      "rdap_endpoint": "https://rdap.example.com/domain/",
      "whois_server": "whois.example.com",
      "whois_enabled": true
    }
  ]
}
```

For TLDs without native RDAP support, use `https://rdap.org/domain/` as a fallback.
